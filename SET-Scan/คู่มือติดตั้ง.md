# คู่มือติดตั้ง SET Dashboard

## สิ่งที่ต้องมี

- Windows 10 / 11
- Python 3.10 ขึ้นไป
- Internet (สำหรับดึงข้อมูลหุ้น)

---

## ขั้นตอนที่ 1 — ติดตั้ง Python

1. เปิดเว็บ [python.org/downloads](https://www.python.org/downloads/)
2. กด **Download Python 3.x.x** (เวอร์ชันล่าสุด)
3. เปิดไฟล์ที่โหลดมา
4. **สำคัญ** — ติ๊กถูกที่ **"Add Python to PATH"** ก่อนกด Install

   ![Add to PATH](https://i.imgur.com/example.png)

5. กด **Install Now** และรอจนเสร็จ

---

## ขั้นตอนที่ 2 — วางไฟล์โปรแกรม

คัดลอกโฟลเดอร์ **SET_Dashboard** ไปไว้ที่ไหนก็ได้ เช่น `C:\SET_Dashboard\`

ตรวจสอบว่ามีไฟล์เหล่านี้ครบ:

```
SET_Dashboard/
├── start.bat
├── app.py
├── set_data_fetcher.py
├── set_dashboard.html
├── listedCompanies_en_US.xlsx
└── requirements.txt
```

---

## ขั้นตอนที่ 3 — เปิดโปรแกรม

ดับเบิ้ลคลิกที่ **`start.bat`**

- ครั้งแรก: โปรแกรมจะติดตั้ง library อัตโนมัติ รอสักครู่
- Browser จะเปิดขึ้นมาที่ `http://localhost:5000`
- หน้าต่างสีดำ (Terminal) ต้องเปิดทิ้งไว้ตลอดเวลาที่ใช้งาน

---

## ขั้นตอนที่ 4 — ดึงข้อมูลครั้งแรก

1. กดปุ่ม **⟳ Refresh** มุมบนขวา
2. รอประมาณ **5–10 นาที** (ดึงข้อมูลหุ้นกว่า 900 ตัว)
3. เมื่อเสร็จ ข้อมูลจะแสดงครบทุกหน้า

---

## การใช้งานทั่วไป

| สิ่งที่ต้องทำ | วิธี |
|---|---|
| เปิดโปรแกรม | ดับเบิ้ลคลิก `start.bat` |
| ปิดโปรแกรม | ปิดหน้าต่างสีดำ (Terminal) |
| อัปเดตข้อมูล | กด **⟳ Refresh** ในเว็บ |
| เปิดเว็บใหม่ | พิมพ์ `localhost:5000` ใน browser |

**แนะนำให้ Refresh ช่วง 17:30–23:00 น.** หลังตลาดปิดและ Yahoo Finance sync ข้อมูลเสร็จแล้ว

---

## แก้ปัญหาเบื้องต้น

**เปิด `start.bat` แล้วปิดเองทันที**
- คลิกขวาที่ `start.bat` → Open with → Notepad ตรวจดูว่าไฟล์ครบ
- ลองเปิด Command Prompt แล้วพิมพ์ `python --version` ถ้าขึ้น error แสดงว่า Python ยังไม่ได้ติดตั้งหรือไม่ได้ Add to PATH

**Browser ขึ้น "This site can't be reached"**
- รอสัก 5 วินาทีแล้วกด Refresh ใน browser
- ตรวจว่าหน้าต่างสีดำยังเปิดอยู่

**กด Refresh แล้วข้อมูลไม่มา / error**
- รอ 30 นาทีแล้วลองใหม่ (Yahoo Finance อาจ rate limit ชั่วคราว)
- ข้อมูลเดิมจะยังแสดงอยู่ระหว่างรอ

**ต้องการอัปเดตรายชื่อหุ้น**
- โหลดไฟล์ใหม่จาก [set.or.th](https://www.set.or.th/th/market/product/stock/quote/all-stock) แล้วแทนที่ `listedCompanies_en_US.xlsx`
